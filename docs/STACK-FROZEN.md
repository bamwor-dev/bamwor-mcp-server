## Build Instructions

### Prerequisites
- Node.js >= 20
- npm >= 9

### Build
```bash
cd mcp/
npm ci          # install dependencies
npm run build   # compile TypeScript → dist/
npm run lint    # type-check without emitting
```

### Verify build
```bash
head -1 dist/index.js
# Expected: #!/usr/bin/env node

npm pack --dry-run
# Expected: ~39 files, ~13KB compressed
# Must NOT contain: src/, .env, Dockerfile, docker-compose.yml
```

## Test Instructions

### Test 1: Server starts
```bash
BAMWOR_API_KEY=your_key_here node dist/index.js
# Expected: "Bamwor MCP Server started (stdio transport)" on stderr
# Then server waits for MCP JSON-RPC on stdin
# Press Ctrl+C to exit
```

### Test 2: Validate tools with Claude Desktop
1. Add to `claude_desktop_config.json`:
```json
{
  "mcpServers": {
    "bamwor-world-data": {
      "command": "node",
      "args": ["/absolute/path/to/mcp/dist/index.js"],
      "env": {
        "BAMWOR_API_KEY": "your_key_here"
      }
    }
  }
}
```
2. Restart Claude Desktop
3. Test each tool:
   - "What is the population of Brazil?" → triggers `get_country`
   - "Search for countries with 'south' in the name" → triggers `search_countries`
   - "Find cities named Tokyo" → triggers `search_cities`
   - "List the 5 largest cities in Germany" → triggers `list_cities_by_country`
   - "Compare France and Germany" → triggers `compare_countries`

### Test 3: Docker (isolated)
```bash
cd mcp/
cp .env.example .env
# Edit .env → set real BAMWOR_API_KEY

docker compose build
docker compose run --rm -i bamwor-mcp
# Expected: server starts on stdio
# Ctrl+C to exit
```

### Test 4: npx simulation
```bash
cd mcp/
npm pack
# Creates bamwor-mcp-server-0.1.0.tgz

BAMWOR_API_KEY=your_key npx ./bamwor-mcp-server-0.1.0.tgz
# Expected: server starts
```

## Isolation Guarantees

| Check | Status |
|-------|--------|
| Own Docker network (`bamwor-mcp-network`) | Yes |
| No ports exposed | Yes |
| No shared volumes | Yes |
| No direct DB access | Yes |
| API-only via public HTTPS | Yes |
| Non-root container user | Yes (`mcpuser:1001`) |
| Memory limit | 64MB |
| CPU limit | 0.25 |
| Log rotation | 5MB x 3 files max |
| No references to other stacks | Yes |
| No references to other networks | Yes |

## Tools Reference

| # | Tool | Endpoint Used | Description |
|---|------|---------------|-------------|
| 1 | `get_country` | `GET /countries/{slug}` | Country detail with stats |
| 2 | `search_countries` | `GET /search?type=country&q=` | Search countries by name |
| 3 | `search_cities` | `GET /search?type=city&q=` | Search cities by name |
| 4 | `list_cities_by_country` | `GET /countries/{slug}/cities` | List cities in a country |
| 5 | `compare_countries` | `GET /countries/{slugA}` + `GET /countries/{slugB}` | Compare two countries |
